## How to run the Project

### Prerequisites
- Docker
- Docker Compose

Clone the repository:
```
git clone https://github.com/jteoni/business-rules-plataform.git
cd business-rules-plataform
```
Start docker-compose:
```
docker-compose up
```
Access the aplication:
```
http://localhost:8000
```

## How to test

Backend:
```
python manage.py test
```
Frontend:
```
yarn test
```
